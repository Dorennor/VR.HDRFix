# VR.HDRFix

## About This Project

This project is a rework of the original [`hdrfix`](https://github.com/bvibber/hdrfix) tool by [Brion Vibber](https://github.com/bvibber), which was written in Rust. I created this .NET version mainly for personal use, as I often encountered stability issues with the original tool—especially when files were locked or in use by applications like Nvidia screen recorder or screenshot tools.

Since I am more comfortable with .NET than Rust, and was unable to fix the original code, I decided to rewrite the tool in a language I know better. My goal was to make it more reliable for my own workflow, especially in handling file access issues.
